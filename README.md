# Зорин Алексей ЭФБО-08-24 Вариант 6

## Конвертер температур
 Возможности:
 Мгновенная конвертация между °C и °F
 Адаптивный дизайн для всех устройств
 Интуитивный интерфейс с теплым фиолетовым дизайном
 Реальное время - изменения происходят сразу при вводе
 Автоочистка при некорректном вводе
 Быстрый старт

Предварительные требования
Node.js (версия 14 или выше)
npm или yarn


### Основное использование
Ввод температуры в Цельсиях:

Кликните на поле "Градусы Цельсия (°C)"

Введите числовое значение

Значение в Фаренгейтах обновится автоматически

Ввод температуры в Фаренгейтах:

Кликните на поле "Градусы Фаренгейта (°F)"

Введите числовое значение

Значение в Цельсиях обновится автоматически

### Частые проблемы
Приложение не запускается:

Убедитесь, что Node.js установлен

Проверьте версию: node --version

Удалите папку node_modules и выполните npm install заново

Поля не обновляются:

Проверьте, что вводите числовые значения

Убедитесь, что JavaScript включен в браузере

Некорректные расчеты:

Используйте точку как разделитель дробей: 36.6 (не 36,6)

Не вводите буквы или специальные символы
